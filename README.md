## 📱  App statement

#### ✅ 서비스명 : Seoullo <br>

Seoullo는 국내 외국인의 성공적인 서울 안착을 위한 정보를 제공하는 어플은 어떨까? 라는 생각에서 시작하게 되었습니다. 서울 열린 데이터광장의 공공 API를 활용해 외국인들에게 필요한 정보들과 한국어 퀴즈를 제공합니다.
